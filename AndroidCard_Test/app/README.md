##### 1,不要随意占用用户的存储空间；
##### 2，不要随意在外置内存卡上创建文件夹或是文件，要在自己App对应的文件夹下，缓存或保存响应的数据，卸载的时候数据都会被擦除掉；
##### 3，对占用的磁盘空间有上限，并按照一定策略进行清除；
